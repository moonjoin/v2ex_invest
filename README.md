# V2EX投资帖子阅读器

## 项目简介

V2EX投资帖子阅读器是一个基于PyQt5开发的桌面应用程序，专门用于抓取和整理V2EX投资板块的帖子内容。该工具可以帮助用户高效地浏览和分析V2EX投资相关的讨论帖子。

## 主要功能

### 1. **智能内容抓取**
- 自动获取V2EX投资板块的RSS订阅内容
- 支持自定义日期范围筛选帖子
- 定时自动刷新功能，实时获取最新投资讨论

### 2. **多格式导出**
- **HTML格式导出**：生成易于阅读的网页文件，方便在浏览器中查看
- **AI优化JSON导出**：专为AI阅读和分析设计的结构化数据格式

### 3. **智能分析功能**
- **关键点提取**：自动提炼帖子的核心内容
- **情感分析**：分析帖子的情感倾向
- **标签提取**：自动生成相关话题标签
- **内容摘要**：提供帖子的简明概述

### 4. **用户友好界面**
- 直观的图形化界面
- 灵活的日期选择器，支持自定义时间范围
- 可配置的导出选项

## 适用场景

- 投资爱好者跟踪V2EX社区的投资讨论
- 收集和归档特定时间段的投资相关话题
- 利用AI工具分析投资趋势和观点
- 建立个人投资信息知识库

## 技术特点

- 使用Python + PyQt5开发，跨平台支持
- 结构化数据输出，便于后续处理和分析
- 包含元数据管理，方便追溯和管理导出内容

这是一个实用的信息聚合工具，特别适合需要系统性收集和分析V2EX投资板块内容的用户。
